# 🚀 Personal Portfolio Website

A modern, dynamic **full-stack portfolio website** built using Django, designed to showcase my projects, skills, and journey as a developer.

---


## 📌 About the Project

This portfolio is not just a static website — it is a **fully dynamic system** where content can be managed through the backend.

It is built with scalability, clean UI, and real-world practices in mind.

---

## ⚙️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite (Development)
* **Media Handling:** Django Media Files
* **Version Control:** Git & GitHub

---

## ✨ Features

* 🔹 Dynamic Projects Section (Admin Controlled)
* 🔹 Blog System with Full CRUD
* 🔹 Contact Form with Backend Handling
* 🔹 Responsive UI (Mobile + Desktop)
* 🔹 Media Upload Support
* 🔹 Clean and Modular Code Structure

## ⚡ Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/Personal-Portfolio.git
cd Personal-Portfolio
```

### 2️⃣ Create virtual environment

```
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the server

```
python manage.py runserver
```
